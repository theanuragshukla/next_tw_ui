## Getting Started

- Clone the repo
```
https://github.com/theanuragshukla/next_tw_ui
```
- Change directory to this project
```
cd next_tw_ui
```
- Install dependencies
```
npm install
```
- Run the dev server
```
npm run dev
```
- Visit [`http://localhost:3000`](http://localhost:3000) to view the page
